# Savvy Stationery 
This is an online shopping application that allows users to browse for a selection of stationery and choose which to buy. 

# Technologies Used
* C#
* PostgreSQL
* EntityFramework Core
* ASP.NET Core
* HTML
* CSS
* AJAX 
* HTTPClient
* Serilog 
  
# Features
Customers are able to:
  * View items
  * Select a location
  * Put items into a cart
  * View their cart
